Discord Moderation Training

Part 1: Discord Basics

The in-charge trainer should ask the member if they know how to copy a user ID.

The in-charge trainer should ask the member if they know how to activate 2FA.

The in-charge trainer should ask the member if they know how to access the audit logs in a server.

1. Warn Command

Command: ?warn @user [reason]

This command is used to warn a user when they have broken a minor rule. The user will get a private message with your warning.

2. Mute Command

Command: ?mute @user [reason]

This command is used to mute a user. It will prevent them from typing in your channels. A time limit may be added but it can also be permanent. For more information about the time limit, look out the website of the bot you are using.

3. Kick Command

Command: ?kick @user [reason]

The kick command is used to remove a user from the server. However, they can still join back using an invite link.

4. Purge Command

Command: ?purge @user [reason]

The purge command is used to delete a number of messages from a channel. Question: What would be a reason for purging a given amount of messages?

5. Ban Command

Command: ?ban @user [reason]

The ban command is used to remove the user from the server permanently. They will be unable to join unless unbanned. The messages of the user will also be deleted. They are still able to join back with a Proxy or VPN.

6. Unban Command

Command: ?unban [UserID]

The ban command is used to remove the user from the server permanently. They will be unable to join unless unbanned. The messages of the user will also be deleted.
